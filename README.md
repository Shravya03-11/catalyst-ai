# CatalystAI

CatalystAI is an AI-powered platform for accelerating catalyst and synthetic biology discovery for sustainable fuel production.

## Features
- Reaction discovery input
- AI-generated catalyst candidates
- Ranking and performance visualization
- 3D molecular reaction viewer
- Feedback loop with prediction vs actual comparison
- Exportable discovery report

## Workflow
1. Enter target reaction
2. Retrieve known candidates
3. Generate novel AI candidates
4. Rank based on performance
5. Visualize molecules
6. Log experimental results
7. Improve future predictions

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- 3Dmol.js

## Future Scope
- Real scientific database integration
